        Create subscription
        Protocol → Email

5) Attach SNS to CloudWatch
        Search : Amazon CloudWatch
       
        Alarms
        Create alarm
        Select metric
        EC2
        Per-Instance Metrics
        👉 CPUUtilization
        Click Select metric
  
        Fill like this:

                Threshold type → Static
                Whenever CPU is → Greater than 70
                Period → 5 minutes

                👉 Meaning:
                If CPU > 70% → alarm triggers
        
        Click “Additional configuration”
        Datapoints to alarm → 2 out of 2 (or 2 out of 3)
